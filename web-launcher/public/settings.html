üßÆ How Route Planning Works

‚ñ∂ Show algorithm details
1

Calculate Time Budget

Each vehicle has a maximum daily capacity based on your work schedule settings (Day End - Day Start).

2

Estimate Travel Time

The system calculates straight-line distance using the Haversine formula, then estimates travel time using your Average Travel Speed setting, plus the Traffic Buffer percentage.

3

Greedy Nearest-Neighbor Assignment

For each vehicle, the algorithm finds the nearest unassigned booking, calculates if it fits within the time budget (travel + service time), and assigns it. This repeats until all vehicles are at capacity or all bookings are assigned.

4

Optimize with Google Routes

Once bookings are assigned to vehicles, the Google Routes API optimizes the stop order for each route to minimize total travel time.

5

Unassigned Bookings

If a booking cannot fit into any vehicle's time window, it will be flagged as "could not be fit". Try adjusting your schedule times or travel speed settings.
